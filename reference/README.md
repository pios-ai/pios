# Reference Implementations

Minimal, runnable code examples that demonstrate the patterns described in the [methodology docs](../docs/).

Each subdirectory is a self-contained example with its own README explaining what it demonstrates and how to run it.

## Planned examples

- `minimal-system/` — The smallest possible PiOS: one pipeline + one agent + cron
- `health-pipeline/` — Apple Health data ingestion with daily report generation
- `daily-digest-agent/` — LLM-powered daily briefing from multiple data sources
- `vault-indexer/` — SQLite FTS5 indexing for a Markdown vault
