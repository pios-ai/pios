===============================================================================
DATABASE SCHEMA
===============================================================================

documents:
  - id (TEXT PRIMARY KEY)
  - source (TEXT NOT NULL)
  - type (TEXT NOT NULL)
  - date (TEXT)
  - title (TEXT)
  - tags (TEXT - comma separated)
  - file_path (TEXT NOT NULL UNIQUE)
  - content_hash (TEXT)
  - created_at (TEXT NOT NULL)
  - updated_at (TEXT NOT NULL)
  Indices: source, type, date, created_at

plugin_runs:
  - id (TEXT PRIMARY KEY)
  - plugin_name (TEXT NOT NULL)
  - started_at (TEXT NOT NULL)
  - finished_at (TEXT)
  - status (TEXT NOT NULL) - running|success|failed
  - documents_created (INTEGER)
  - error_message (TEXT)
  - duration_ms (INTEGER)
  - created_at (TEXT NOT NULL)
  Indices: plugin_name, status

plugin_state:
  - plugin_name (TEXT PRIMARY KEY)
  - last_run (TEXT)
  - last_run_status (TEXT)
  - state_data (TEXT - JSON)
  - updated_at (TEXT NOT NULL)

===============================================================================
RUNNING THE PROJECT
===============================================================================

Local Development:
  1. make install              # Install dependencies
  2. make dev                  # Start backend (port 8000)
  3. cd frontend && npm run dev # Start frontend (port 5173)
  4. Open http://localhost:5173

Docker:
  1. make docker-build
  2. make docker-up
  3. Open http://localhost:5173

API Documentation:
  - http://localhost:8000/docs (Swagger UI)
  - http://localhost:8000/redoc (ReDoc)

CLI:
  python cli/pios_cli.py --help
